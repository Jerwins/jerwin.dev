| Time         | Notation              |
| ------------ | --------------------- |
| ------------ | -------------         |
| Constant     | O(1)                  |
| Logarithmic  | O(log(n))             |
| Linear       | O(n)                  |
| Linearithmic | O(nlog(n))            |
| Quadratic    | O($n^2$)              |
| Cubic        | O($n^3$)              |
| Exponential  | O($b^n$), where b > 1 |
| Factorial    | O(n!)                 |
